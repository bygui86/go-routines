
# Sample project to understand how to group go routines

## Run

### No groups
```
go run nogroups.go
```

### Default WaitGroup
```
go run waitgroup.go
```

### Default ErrorGroup
```
go run waitgroup.go
```

### Default ErrorGroup with context
```
go run withctx.go
```
